Btree index:
Сильно ускоряет поиск по числовым значениям и значительно ускоряет поиск по полному совпадению строки. Не подходит для поиска по частичному совпадению строки.

Hash index:
Значительно ускоряет поиск по полному совпадению строки и по числовому значению. Не подходит для поиска по частичному совпадению строки.
